# storybook

## Environment

- Node -v > 12

## Setup
- run `yarn install`

Para instalar los githooks en el proyecto:
- run `pip install pre-commit`
- run `pre-commit install`

## Start Project

- run `yarn start`

## Run test 

- run `yarn test`

### Watch test

- run `yarn test:watch`


## Test Coverage

- run `test:coverage`
